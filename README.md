# AOE2-McMinimap (archived)

**This Python library has been superseded by [aoe2-mcminimap](https://github.com/UnluckyForSome/aoe2-mcminimap).**

The JavaScript module is used by [Pages-AOE2Museum](https://github.com/UnluckyForSome/Pages-AOE2Museum) for in-browser isometric minimap rendering via `shared-module-service.js`.

This repo is the legacy Python/Pillow implementation (PyPI: `aoe2-mcminimap`). It remains useful only as a parity-harness oracle via `sourcemodules/` until `build/parity/oracle-minimap.py` is retired.
